## 📌 Introduction
**Freedom Eyes**는 파이브엠 및 게임 내 거래 시 발생할 수 있는 사기 피해를 원천 차단하기 위해 개발된 윈도우 전용 데스크톱 보안 애플리케이션입니다. 

클립보드에 복사된 이미지를 Windows Native OCR 엔진으로 0.1초 만에 스캔하여, 대상의 고유번호(UID)를 프리덤 데이터베이스와 대조하고 실시간으로 위험 여부를 화면에 오버레이(Overlay)로 알려줍니다. 무거운 게임 환경에서도 프레임 드랍 없이 깃털처럼 가볍게 동작하도록 완벽하게 최적화되었습니다.

<br>

## 🔒 보안 및 인증 완료 (Security Verified)
Freedom Eyes는 유저의 안전과 보안을 최우선으로 설계되었습니다.

- **✅ 클린 소프트웨어 인증:** 악성코드, 해킹툴, 스파이웨어가 전혀 포함되어 있지 않은 안전한 애플리케이션입니다.
- **✅ 메모리 비간섭 (Anti-Cheat Safe):** 게임 클라이언트의 메모리를 변조하거나 후킹(Hooking)하지 않습니다. <br/>오직 '클립보드(복사된 이미지)'만을 판독하므로 서버 밴(Ban)이나 안티치트 제재의 위험이 **0%** 입니다.
- **✅ 개인정보 보호:** 사용자의 게임 로그인 정보나 개인 식별 정보를 절대 수집하지 않습니다.
- **✅ 암호화 통신:** 모든 통신 및 하드웨어(HWID) 인증은 종단간 암호화(SSL/TLS)를 통해 안전하게 처리됩니다.

<br>

## ✨ 핵심 기능 (Key Features)
* **Zero-Click 실시간 감지:** 캡처(`Win+Shift+S`)하는 순간 백그라운드에서 자동으로 판독 및 결과 송출.
* **초경량 엔진 탑재:** 무거운 AI 모듈을 버리고 윈도우 내장 API를 사용하여 CPU 점유율 최소화 (50MB 미만).
* **스마트 티커 (Ticker):** 게임 화면을 가리지 않는 세련되고 모던한 알림 UI 제공.
* **Pro 멤버십 지원:** 1PC 기기 귀속(HWID) 보안 시스템, 최대 5인 동시 다중 스캔 및 상세 전과 리포트 제공.
